TicketReservationSystem

This is an open source framework for booking ticket .

Setting up the environment

You should have following software install in your machine.

  • Java 7 or above
  • Maven 3 or above
  • Git
  • Any IDE like eclipse

How to run the application

  • Clone the code from repository using git clone
  • Move to the right branch.(currently develop1.0RC) using git checkout develop1.0RC
  • Go inside the TicketReservationSystem folder
  • Clean the workspace using mvn clean
  • Install the maven dependency using mvn install. It'll pull all the dependency
  • Go to booking-site folder run the jetty server (which is embedded currently) using mvn jetty:run
  • type http://localhost:8080/api/bookings . You'll get a response saying Accepted.
